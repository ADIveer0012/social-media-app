# TASK 2 - Social Media Platform

A mini social media platform built using:

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js
- MongoDB

Features:
- User Registration & Login
- JWT Authentication
- Create Post
- View Feed
- Like Posts
- Follow / Unfollow Users
- User Profiles

How To Run:

1. Backend
cd backend
npm install
npm run dev

2. Frontend
Open frontend/feed.html in browser

Environment Variables (.env inside backend)

MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
